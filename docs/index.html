SMTP Client Wrapper

The goal of this SMTP Client Wrapper is to utilize the adapter pattern to abstract SMTP functionality.

Notes:

  • This project wraps .NET's SmtpClient.
  • .NET's SmtpClient offered more features than I needed for a few applications, and I didn't want to be locked into a single implementation.
  • With this wrapper, SMTP client implementations can be swapped out as needed by creating new adaptees/adapters.
